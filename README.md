# ikaros-web
[ikaros](https://github.com/li-guohao/ikaros) 项目的前端后台界面，已经编译进 [ikaros](https://github.com/li-guohao/ikaros) 对应的目录中。

基于[ant-design-vue-pro](https://github.com/vueComponent/ant-design-vue-pro)


# 环境和依赖
- node
- yarn
- webpack
- eslint
- @vue/cli
- ant-design-vue@1.x

# 项目下载和运行

## 安装依赖
``` shell
yarn install
```

## 开发模式运行

``` shell
yarn run serve
```

## 编译项目
``` shell
yarn run build
```


## Lints and fixes files
``` shell
yarn run lint
```
